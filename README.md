# Muwemi Ndovie - Portfolio Website

A dynamic Django portfolio website showcasing mechanical engineering projects, skills, and achievements.

## Features

- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Dark/Light Mode**: Toggle with localStorage persistence
- **Dynamic Content**: Admin-editable skills, projects, and content
- **Contact Form**: Saves messages to database and sends email notifications
- **Project Showcase**: Detailed project pages with image carousels
- **Blog Ready**: Markdown support for future blog posts

## Tech Stack

- **Backend**: Django 4.2
- **Frontend**: Tailwind CSS, Alpine.js
- **Database**: SQLite (development), PostgreSQL (production)
- **Deployment**: PythonAnywhere ready
